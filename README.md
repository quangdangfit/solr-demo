# solr-demo

### Overview:
Solr demo by Golang

### How to install packages:
- ```go mod init```
- ```go mod tidy```
- Install to vendor ```go mod vendor```
- Install single package ```go get github.com/path/to/package@<branch-name>```

### How to run:
- cp config/.env_sample.sh config/.env.sh
- source config/.env.sh
- go run main.go
